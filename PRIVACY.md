# Privacy Policy - Meeting Mode Writer

**Last Updated:** January 2025

## 1. Information We Collect

Meeting Mode Writer collects the following information to provide its services:

### 1.1 Google Docs Access
- Document content and structure
- Document metadata (title, creation date)
- Document formatting and styles

### 1.2 User Information
- User email address (for document ownership)
- User preferences and settings

### 1.3 Configuration Data
- Template definitions and customizations
- Action item data and status
- Meeting metadata and history

### 1.4 Activity Logs
- Timestamps of template applications
- Formatting operations performed
- Export activities

## 2. How We Use Your Information

We use your information to:
- Apply templates to documents
- Format meeting content
- Track action items and tasks
- Generate meeting summaries
- Export documents to various formats

## 3. Data Storage and Retention

### 3.1 Storage Location
- All configuration data is stored in Google Apps Script PropertiesService
- Templates and action items are stored with the document
- Data is stored in your Google account's document properties

### 3.2 Data Retention
- Configuration data is retained until you delete it
- Activity logs are limited to the most recent 100 entries
- Document data remains in Google Docs until you delete it

## 4. Data Sharing

We do not sell, rent, or share your data with third parties for marketing purposes.

### 4.1 Limited Sharing
- No data sharing occurs
- All data remains within your Google account

## 5. Data Security

We implement appropriate security measures:
- All data is stored within Google's secure infrastructure
- OAuth 2.0 authentication for Google API access
- Document access is limited to documents you own
- No external servers or databases

## 6. Your Rights

You have the right to:
- Access your templates and settings
- Delete custom templates and action items
- Clear activity logs
- Uninstall the add-on at any time

## 7. Children's Privacy

Meeting Mode Writer is not directed to children under 13. We do not knowingly collect personal information from children under 13.

## 8. Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new policy on this page.
